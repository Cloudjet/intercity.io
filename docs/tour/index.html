What's included in Intercity

Host all the apps

Deploy and host Ruby on Rails, Python, Node.js and PHP apps within minutes on your own server of choice.

No more hassle

Get a server at your favorite cloud server, VPS, or hardware vendor and connect it to your Intercity dashboard. Intercity takes care of installing and configuring it to host your apps.

Learn more about installing a server →

Deploying is a breeze

Just add an app to a server via the Intercity dashboard. Then add a git remote to your repository and git push to deploy.

Learn more about deploying →

$ git remote add dokku dokku@37.139.13.229:intercity-io
$ git push dokku master
